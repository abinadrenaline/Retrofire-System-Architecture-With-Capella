# Retrofire System Architecture - MBSE using Capella and ARCADIA
Disclaimer: The Model depicted below is intended to provide inspiration and may not represent the current true Architecutre of WARR Rocketry's Project Retrofire.
## Framework - 
### with Parallel Requirements Capture and Assignment in all layers - 
### 1. Operational Analysis - Operational Entities, Operational Capabilities, Operational Architecture
### 2. System Analysis - Mission Breakdown, Capability Scenario Definition using System Functions
### 3. Logical Architecture - System Function Breakdown to Logical Functions, Logical Architecture 
### 4. Physical Architecture - Assigning Physical Elements to Logical Processes
